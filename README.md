# blogApp
This is my small MVC project. It has both backend and frontend part. Technologies that were used:
+ Spring Boot
  + Spring Security
  + Spring Data JPA
  + Java Mail Sender
+ Flyway
+ Thymeleaf
+ Bootstrap
